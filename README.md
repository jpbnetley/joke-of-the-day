# Joke of the day

This app pulls a random joke from `reddit.com/r/jokes`

## Getting started

This project requires [node](https://nodejs.org/en/download/) to be installed.

### Install dependencies

```bash
npm i
```

### Running th project

```bash
npm run start
```
